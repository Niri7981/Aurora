Rust 程序的入口函数。
你运行：

cargo run

最先执行的就是 main()。

std::env::args().nth(1)

意思是：读取命令行参数里的第一个用户参数。

比如：

cargo run ./my-workspace

参数大概是：

第 0 个参数：程序本身
第 1 个参数：./my-workspace

所以它的类型是：

Option<String>

f let Err(err) = ...
if let Err(err) = aurora::run(...) {

这是 Rust 里很常见的错误处理写法。

eprintln!
eprintln!("错误：{err}");

println! 是打印到正常输出。
eprintln! 是打印到错误输出。

一般错误信息用 eprintln!。


std::process::exit(1);

意思是：直接退出程序。

0 一般表示正常退出。
1 一般表示异常退出。
